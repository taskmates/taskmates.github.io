Some things you can do with Taskmates

  • Ask your taskmates to read, update, and write code, run unit tests and fix them, run CLI tools, interpret images, and read PDFs.
  • Ask your taskmates to add missing unit tests, convert code to use a different library, language, architecture and more.
  • Interactively work with your taskmates in any Markdown document, directly from IntelliJ (or RubyMine, PyCharm, WebStorm, etc.).
  • Use the Taskmates CLI to ask your taskmates to tasks that involve interaction with other CLI tools, APIs etc
  • Create your own taskmates with custom knowledge, different purposes, and different integrations.
